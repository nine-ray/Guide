## 统一返回数据格式
| 字段名        | 参数含义     | 类型   | 备注 |
| ------------- | ------------ | ------ | ---- |
| success            | 请求是否成功       | true/false |      |
| code     | 状态吗   | string |      |
| data     | 请求返回数据  | JSON   |      |
| message      | 请求返回消息 | string |      |

## 统一返回错误码

| ------------- | ------------ | 
| 6009            | 未知错误       | 
| 6008     | token无效   | 
| 6001     | 注册手机号已经被使用  |
